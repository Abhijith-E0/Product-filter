# Product Filter System

A simple and responsive **product filtering system** built using **HTML, CSS, and JavaScript**.  
Users can filter products based on categories like Phones, Laptops, and Watches without reloading the page.

## Features

- Category-based filtering
- Dynamic product rendering using JavaScript
- Responsive card layout
- Clean UI design
- No external libraries required
- Fast client-side filtering

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
